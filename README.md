# Meme A Day Studio Site

Public support and privacy pages for Meme A Day Studio.

This repository intentionally contains only static public pages:

- `/privacy/`
- `/support/`

## GitHub Pages

Create this repository as public on GitHub, then enable GitHub Pages:

1. Open repository Settings.
2. Go to Pages.
3. Set Source to "Deploy from a branch".
4. Set Branch to `main`.
5. Set Folder to `/root`.
6. Save.

Expected URLs:

- `https://lambie-lanman.github.io/memeaday-studio-site/privacy/`
- `https://lambie-lanman.github.io/memeaday-studio-site/support/`
